AIDEEP 사용법

노드란?

그래프 위의 동그라미·네모 하나하나가 노드입니다. 각각 하나의 주제를 담는 단위예요.

중심 주제 노드 — 그래프의 중심이 되는 주제
일반 주제 노드 — 중심 주제에서 뻗어나가는 하위 주제
  • 빈 공간 우클릭중심 주제 노드 생성
  • 빈 공간 더블 클릭일반 주제 노드 생성
  • 노드를 다른 노드 위로 드래그그 노드의 하위로 이동
  • Shift + 드래그여러 노드를 한번에 이동
  • Option(Alt) + 드래그노드 하나만 이동
